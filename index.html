About

Hi there! I'm a full-stack web developer with experience working in Ruby on Rails, JavaScript, and React.js/Flux. Coding fascinates me as both an art and a science--I love that it can be as much an avenue for personal expression as a tool to bring people information, but I also love the intellectual challenges it presents. Each bit of code is a little logic problem for you to solve with the tools at your disposal, and each day is an opportunity to learn more and thus gather more tools to solve them with. When I'm not coding I'm hanging out with my amazing wife, competing in trivia nights, immersing myself in literature, music, and cinema, pondering the aesthetic possibilities of noise, and resenting my cat for tearing up my couch.
